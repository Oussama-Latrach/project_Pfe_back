 DGCNN Point Cloud Classification

 This project implements DGCNN for classifying LiDAR point clouds.

 ## Installation
 ```bash
 pip install -r requirements.txt
 ```

 ## Training
 ```bash
 python train.py
 ```

 ## Inference
 ```bash
 python inference.py
 ```

 ## Configuration
 Hyperparameters and paths are in `config.yaml`.
